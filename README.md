# Auto MacNet

## About

Automatically init network preferences based on network SSID change

## Usage

## Requirement

- Give permission to `/usr/bin/security` the ssid of the hotspot you want to connect at *Keychain Access* -> *ssid* -> *Get Info* -> *Access Control*
